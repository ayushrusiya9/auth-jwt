# Backend JWT Authentication API (Django + DRF)

JWT Authentication implementation using **Django** and **Django REST Framework**.  
This project provides backend APIs for user signup, login, JWT token generation, protected routes, and auth flow for frontend consumption (e.g., React).

---

## Project Overview

This repository implements a **practice backend authentication API** with JSON Web Tokens (JWT).  
It includes secure user registration, login, token generation, and protected endpoints using Django REST Framework.


---

## Tech Stack

| Category | Technology |
|----------|------------|
| Backend framework | Django |
| API framework | Django REST Framework (DRF) |
| Auth | JWT (via Simple JWT or custom implementation) |
| Database | SQLite (default) |
| Dev tools | Python, pip, Django Admin |
